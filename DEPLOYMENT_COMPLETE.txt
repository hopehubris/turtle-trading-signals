TROUBLESHOOTING QUICK REFERENCE

  → Check: pm2 logs turtle-trading-signals --err
  → Try: npm run build && pm2 restart turtle-trading-signals

  → Check: curl http://192.168.1.51:3001/api/health

  → Restart: pm2 restart turtle-trading-signals
